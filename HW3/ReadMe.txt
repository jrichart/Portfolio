========================================================================
    CONSOLE APPLICATION : 343HW3 Project Overview
========================================================================

This program was developed in VS2013

343HW3.cpp
    This is the main application source file.

The included text file uses the expected syntax to generate output.
Other text files of the same name could be used, provided they
use the same syntax

This program uses Dijkstra's algorithm to determine the shortest path
between places given in the text file. The file includes the number of
places, their names, and a set of three numbers [starting location]
[ending location] [distance].
